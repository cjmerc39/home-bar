# Home Bar

A single-file, mobile-first PWA for running a home bar: bottle inventory,
a spec book of classic cocktails, a live "what can I make right now" matcher
with buy-one-bottle unlock suggestions, a guest-facing Menu Mode — and an AI
shelf scan that reads bottles straight off a photo of the bar.

Everything lives in `index.html` (no build step, no framework). Data persists
in `localStorage` under the `bar-v1` key, with one-tap JSON export/import in
Settings as the backup story.

## Run the tests

```
npm i jsdom
node home-bar.test.js
```

## Deploy the AI scan worker

The photo scan needs a tiny Cloudflare Worker so the Anthropic API key stays
server-side. Full step-by-step instructions are in the comment block at the
top of `home-bar-worker.js` — in short: create a worker named `home-bar`,
paste the file, deploy, and add your `ANTHROPIC_API_KEY` as a secret.

If `SCAN_URL` in `index.html` is set to an empty string, the scan button
disappears and everything else works normally.
